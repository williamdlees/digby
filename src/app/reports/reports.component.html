VDJbase Reports

Some reports run just against AIRR-Seq or genomic data, while some will run on both. Please select above the species and datasets on which you would like to report. You can further refine the samples against which the reports will run by using the filters on the Samples tabs to display just those samples that you wish to be included. To run a report, click on an icon in the Run column corresponding to the output format that you would like (e.g. pdf, on-screen, Excel)
